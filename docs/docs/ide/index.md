# Online IDE and Coding Practice

GitHub Pages is a static site, so it cannot execute Java, Python, SQL, or Playwright code directly.

## Recommended Now

Use GitHub Codespaces:

1. Open the GitHub repo.
2. Click **Code**.
3. Click **Codespaces**.
4. Create a new Codespace.
5. Run practice commands in the browser terminal.
